# Flask Calculator Project

## Project Description
A simple calculator web application built using Python Flask.
It demonstrates OOP concepts and basic frontend–backend communication.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Features
- Addition, Subtraction, Multiplication, Division
- Button-based calculator UI
- Backend calculation using Flask API
- Responsive design (works on mobile)

## OOP Concept
A Calculator class is used to perform calculations.

## How to Run
1. Install Flask  
   pip install flask
2. Run the app  
   python app.py
3. Open browser  
   http://localhost:5000
